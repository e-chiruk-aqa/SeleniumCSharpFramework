# Selenium .Net Core Automation Framework

.Net Core automation framework to simplify work with Selenium

#### Environment

* .Net Core 3.1

#### Used tools
* Selenium WebDriver 3.141.0
* NUnit 3.12 
* NLog 4.6.8
* SpecFlow 3.1.67
* Allure 2

To view reports locally you need to install `allure-commandline`
